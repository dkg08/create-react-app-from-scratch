
# Project Title

Creating React App from scratch with parcel.js


## Run Locally

Clone the project

```bash
  git clone https://github.com/dkg08/create-react-app-from-scratch.git
```

Go to the project directory

```bash
  cd create-react-app-from-scratch
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
